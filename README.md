# 刘看山的灵感跳跃

一个纯前端 Canvas 横版跳跃小游戏。玩家操作勇士看山收集灵感星，使用雪盾、风翼、冰星等道具，挑战三段冰雪关卡，并和公主看山一起回到冰洞。

## 在线托管

这是静态网页项目，不需要后端服务。可以直接部署到：

- Netlify
- Cloudflare Pages
- GitHub Pages

Netlify 连接 GitHub 仓库时，配置可以留空：

- Build command: 留空
- Publish directory: `.`

仓库内的 `netlify.toml` 也已经写好了同样的静态发布配置。

## 入口

- 电脑版：`index.html`
- 手机触屏版：`mobile.html`

手机触屏版保留同一套关卡和素材，但使用更大的虚拟按键、横屏布局、防滚动处理和基础 PWA manifest。

## 本地运行

最简单的方法是在项目根目录启动静态服务器：

```powershell
node server.mjs 8000
```

然后打开：

```text
http://127.0.0.1:8000/
```

也可以使用任意静态服务器打开 `index.html`。

## 操作

- 方向键或 `A/D`：移动
- 空格、上方向键或 `W`：跳跃
- 下方向键或 `S`：空中下压
- `J/K`：获得冰星道具后发射冰星
- `P`：暂停
- `Enter`：菜单确认

触屏设备会显示方向、跳跃、下压和冰星攻击按钮。

## 规则

- 每收集 12 颗灵感星增加 1 条命。
- 雪盾可以冻结撞到盾上的怪物。
- 风翼会短时间提高跳跃能力。
- 冰星可以远程攻击怪物。
- 星爆会释放额外灵感星并加分。

## 关卡

- `1-1 海风草坡`：入门关，熟悉跳跃、问号块和下压。
- `1-2 灵感工坊`：平台节奏更密，加入更多墨池和道具。
- `1-3 月光云城`：云台更窄，敌人更快，是最终关。
